# Project 1: Static Webpage Design — DecodeLabs Internship

A static personal developer portfolio website for Muhammad Talha Zubair, a BSCS student and frontend/AI developer. The website showcases his background, skills, and key projects using a responsive, modern dark-themed interface.

## Tech Stack
- HTML5
- CSS3 (Grid & Flexbox)

## Key Features
- **Semantic HTML**: Built using structured, meaningful elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`) with a strict heading hierarchy.
- **Responsive Layouts**: Designed mobile-first, utilizing CSS Grid for macro-layouts and Flexbox for micro-alignments.
- **Modern & Premium Design**: Implemented with smooth transitions, custom scroll behavior, glassmorphism, and a high-contrast dark color palette.
- **Accessibility (a11y)**: Fully keyboard-navigable with visible focus outlines, proper input-label bindings, and color contrast ratios exceeding 4.5:1.
- **BEM Methodology**: Organised styling with clean BEM naming convention class selectors in `style.css`.
- **Cumulative Layout Shift (CLS) Prevention**: Images pre-defined with aspect ratios and fallback setups.
